# Autofocus - Pomodoro Timer

A beautiful, gravity-defying Pomodoro timer with stunning glassmorphism design and smooth animations.

## Features

- **Immersive Animations**: Floating objects, smooth transitions, and physics-based movements
- **Glassmorphism Design**: Beautiful frosted glass effects with backdrop blur
- **Dark Mode**: Toggle between light and dark themes with smooth transitions
- **Flexible Theme System**: Easy to add custom themes (see THEMES.md)
- **Full Pomodoro Functionality**: 25/5/15 minute cycles with automatic breaks
- **Progress Visualization**: Circular progress ring with perfectly aligned countdown
- **Keyboard Shortcuts**: Space to start/pause, R to reset
- **Mouse Interactions**: Floating objects react to cursor proximity
- **Notifications**: Browser notifications and completion sounds
- **Persistent Settings**: Theme preference saved to localStorage

## Usage

Simply open `index.html` in your browser. No build process required!

## Customization

See [THEMES.md](THEMES.md) for detailed instructions on creating custom themes.
